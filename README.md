# Calculadora Geométrica

Calculadora geométrica feita em JavaScript puro para praticar lógica de programação.

## O que calcula
- Área do triângulo
- Área do retângulo
- Área do quadrado
- Área do trapézio
- Área do círculo

## Como usar
Abra o arquivo no navegador e siga as instruções no prompt.

## Tecnologias
- JavaScript
